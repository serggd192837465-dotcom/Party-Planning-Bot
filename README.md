# Бот для планирования вечеринки

## Описание
Учебный курсовой проект по JavaScript: Telegram-бот, который помогает организовать вечеринку.
Бот позволяет вести список гостей, собирать предпочтения, считать бюджет и работать с чек-листом задач.
Проект сделан в модульном стиле (import/export), как требуется в ТЗ.

## Функционал
- Добавление и удаление гостей (имя, email, статус приглашения).
- Просмотр списка гостей.
- Сбор предпочтений гостей: музыка, еда, тематика.
- Подсчет топ-3 популярных предпочтений.
- Установка лимита бюджета.
- Добавление расходов и просмотр остатка бюджета.
- Предупреждение при превышении лимита.
- Добавление задач в чек-лист (название, дедлайн, статус).
- Редактирование, удаление и сортировка задач.
- Фильтрация задач по статусу.
- Сброс списка задач.

## Структура проекта
- `telegram-bot/main.js` — основной файл, запуск бота, обработка команд.
- `telegram-bot/modules/addItem.js` — добавление задачи.
- `telegram-bot/modules/deletItem.js` — удаление задачи.
- `telegram-bot/modules/editItem.js` — редактирование задачи.
- `telegram-bot/modules/sortItems.js` — сортировка задач.
- `telegram-bot/modules/reset.js` — сброс задач.
- `telegram-bot/data/store.json` — локальное хранение данных.

## Технологии
- JavaScript (ES6+)
- Node.js
- Telegram Bot API
- Библиотека `node-telegram-bot-api`
- JSON (для хранения данных)

## Установка и запуск
- Установить Node.js LTS.
- Открыть проект в VS Code.
- Открыть терминал в папке проекта.
- Выполнить команду: `npm install`
- Запустить бота: `node telegram-bot/main.js`
- Открыть бота в Telegram и отправить команды `/start` и `/help`
